# JavaScript Learning

## 1° Lista de Exercícios de JavaScript

Este repositório documenta meu aprendizado em JavaScript, começando pela **lógica de programação** e exercícios básicos.

### Conteúdos da lista
- Variáveis e tipos de dados

- Loops (for, while)
- Exercícios de lógica básica

### Objetivo
Praticar lógica de programação em JavaScript e construir uma base sólida para avançar em tópicos mais avançados.
